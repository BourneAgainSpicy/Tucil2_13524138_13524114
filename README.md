a. Program Voxelizer kami menerima obj file serta maxDepthOctree dan mengembalikan obj file yang telah mengalami voxelization

b. Requirement:
Release GO terbaru: https://go.dev/dl/

c. Cara mengkompilasi dari source code
Jalankan perintah :
go mod init voxelizer

d. Cara menjalankan program dari src code:
Jalankan perintah :
go run ../main.go ../input.obj <maxDepthOctree>
